**Snapycode Mail Your Friend**
This is a wordpress widget plugin.
Using Snapycode mail your friend plugin your site visitor can send emails to their friends or anyone.
<img src ="http://imageshack.us/a/img18/3681/screenshot1qob.jpg" />

== Installation ==

1. Upload snapycode-mail-ur-friend directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activating it navigate to wordpress admin dashboard - Appearance - Widgets, there you will see a widget named snapycode mail ur friend, just drag and drop it in your widget area.
